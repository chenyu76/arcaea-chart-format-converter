# arcaea chart format converter

A program that can simply convert arcaea chart (.aff) files.

**Note**: This progarm only supports Chinese

### 它可以:

- 将两个(或多个)谱面混合起来
- 将谱面中的指定一类元素转换为另一类
    - 将arc转成skyline
    - 将skyline转成arc
    - 将地面note转成skyline
    - 将地面hold转成skyline
    - 将天空note转成skyline
- 在歌曲最前面添加一个note
